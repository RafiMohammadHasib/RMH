# **App Name**: Portfolio Ace

## Core Features:

- Personal Information Display: Showcase the user's name, profession, and a brief bio.
- Skills and Expertise Section: Dedicated section highlighting key skills and areas of expertise, visible with well-placed text.
- Portfolio Showcase: Display of portfolio items with images and brief descriptions.
- Contact Information: Display contact information (email address).
- AI-Powered Project Suggester: An AI tool which analyzes the user's existing portfolio to suggest similar projects to inspire future work.

## Style Guidelines:

- Background color: Very light grey (#F2F4F7) to create a clean and spacious feel.
- Primary color: A muted blue (#6488B1) to convey professionalism and reliability.
- Accent color: Soft green (#8FBC8F) for interactive elements and highlights, providing a subtle contrast.
- Font pairing: 'Playfair' (serif) for headlines to provide elegance and 'PT Sans' (sans-serif) for body text for readability.
- Grid-based layout to create a structured and balanced feel.
- Minimalist icons, mainly line icons, for visual interest and quick comprehension of displayed features and labels.
- Subtle transitions and hover effects to enhance user engagement and highlight interactive elements.
